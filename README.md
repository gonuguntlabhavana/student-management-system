# 🎓 Student Management System

A React Single Page Application (SPA) developed using React and React Router.

## Features

- 🏠 Home Page
- 👨‍🎓 Student Management
- 👩‍🏫 Teacher Management
- 📝 Student Registration
- 📞 Contact Page
- 🔍 Search
- ✏️ Edit
- 🗑️ Delete

## Technologies Used

- React.js
- React Router DOM
- JavaScript
- HTML
- CSS
- Vite

## Installation

```bash
npm install
npm run dev
```
